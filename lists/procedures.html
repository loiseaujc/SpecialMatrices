matmul specialmatrices_tridiagonal Interface

This interface overloads the Fortran intrinsic matmul for a Tridiagonal matrix. The intrinsic matmul is overloaded both for matrix-vector and matrix-matrix products. For a matrix-matrix product , only the matrix has to be a Tridiagonal matrix. Both and need to be standard Fortran rank-2 arrays. All the underlying functions are defined as pure.
